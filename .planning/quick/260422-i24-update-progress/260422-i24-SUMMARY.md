### 修改 `handlePull` 函数添加初始进度显示

**文件:** `src/app/page.tsx`

**问题:**
点击更新按钮后，需要等待后端发送进度事件才能看到进度条启动，这会导致用户感觉有一定的卡顿。

**修改内容:**
在 `handlePull` 函数开始时设置初始进度：
```tsx
const handlePull = async () => {
    if (!config) return;

    setLoading(true);
    setMessage('');
    setProgress({ stage: 'pulling', percent: 5, label: '开始更新...' });

    try {
      const result = await runSmartPull(config);
      // ...
    }
    // ...
};
```

**效果:**
1. 用户点击按钮后会立即看到进度条启动（5%）
2. 文本区域显示 "开始更新..." (Starting update...)
3. 后端的进度事件会覆盖这个初始进度，提供更详细的进度信息
